# FlappyBirdAI
FlappyBird AI (Genetic Algorithms + Neural Networks)

- To train model

`python3 main.py`

- To play with best bird

`python3 run_best_net.py`

- Configuration file: `configs.py`
````python
MUTATE_RATE = 0.8
POPULATION_SIZE = 50
POOL_SIZE = 100
MAX_GENERATIONS = 100
MAX_SCORE = 100

WEIGHT_STDEV = 0
WEIGHT_MEAN = 1

BIAS_STDEV = 0
BIAS_MEAN = 1
````
